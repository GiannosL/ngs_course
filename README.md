# Lecture material for Imputation and Snakemake lectures

This repository stores lecture material for the "Genotype Imputation" and "Snakemake" lectures of Giannos Louloudis for the NGS course in the [Applied Bioinformatics](https://websites.auth.gr/appbio/) (AppBio) MSc programme of the Aristotle University of Thessaloniki.

Here you will be able to find the exercise material for in-class exercises, supplementary material, useful unix commands, and the lecture slides.

Lectures:

* [Genotype Imputation](imputation/)
* [Snakemake](snakemake/)
